Algumas Estrutura de elementos, tags específicas, semântica, etc.


Alguns exemplos de programaçãoem html para começar a aprender

Tag Descrição Atributos Comuns Exemplo de Uso Categoria Semântica
<h1> Título de nível 1 id, class <h1>Bem-vindo</h1> Conteúdo
<a> Link para outra página ou seção href, target, title <a href="https://exemplo.com">Visite</a> Interativo
<img> Imagem incorporada src, alt, width, height <img src="foto.jpg" alt="Foto"> Embutido
<div> Contêiner genérico para agrupar conteúdo id, class, style <div class="caixa">...</div> Não semântico / estrutural
<form> Define um formulário para envio de dados action, method <form action="/envia" method="post">...</form> Interativo
